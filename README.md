# Run

Serve a server in the root of the project:

```
python3 -m http.server 8080
```

And then you can see the project in [localhost:8080](http://localhost:8080/)

# Controls

W, A, S, D : walk

Q, E: rotate

space: jump

C: crouch

shift+move: run

U: open glass door

I: close glass door

O: open yellow door

P: close yellow door

F and SHIFT+F: make time go faster
